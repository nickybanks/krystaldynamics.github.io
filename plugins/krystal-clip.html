KrystalClip

Overview

KrystalClip is a simple and intuitive clipper plugin. It presents 3 different clipping modes to experiment different colors.

Features

  • Smooth, Fuzzy and Hard clipping modes
  • Threshold control
  • Slope control that shapes each curve differently
  • Input and output gain controls
  • Input and output metering system
  • Real-time waveform display
