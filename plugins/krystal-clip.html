KrystalClip

Overview

KrystalClip is a simple and intuitive clipper plugin. It presents 3 different clipping modes to experiment different colors.

Features

  • Smooth, Warm and Hard clipping modes
  • Threshold control
  • Slope control to shape the clipping curve
  • Input/output gain controls
  • Mix control for parallel processing
  • Oversampling up to 4x
  • Real-time waveform display
  • Comprehensive input and output metering
