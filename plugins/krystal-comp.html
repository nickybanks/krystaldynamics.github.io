KrystalComp

Overview

KrystalComp is a precise digital dynamics processor that provides complete control over the compression process. Designed for mixing and mastering, it offers flexible sidechain options and multiple detection modes.

Features

  • Standard compressors controls: Threshold, Ratio, Attack, Release
  • Variable knee control for compression curve shaping
  • RMS and Peak detection modes
  • Input and makeup gain controls
  • Mix control for parallel processing
  • Internal and external sidechain input
  • Sidechain signal equalizer with LP, HP, and bell filters
  • Monitor button for listening to the sidechain
  • Comprehensive input, output, and gain reduction metering
